# Telecom Customer Churn Prediction

## Problem
Telecom companies lose revenue when customers leave (churn).
This project predicts which customers are likely to churn.

## Dataset
Telecom churn dataset with customer demographics and service usage.

## Project Workflow
1. Data cleaning
2. Exploratory data analysis
3. Feature engineering
4. Handling class imbalance
5. Model training
6. Evaluation

## Models Used
- KNN
- SVC
- Logistic Regression
- Random Forest
- Decision Tree

## Evaluation Metrics
- Precision
- Recall
- F1 Score

## Results
Random Forest performed best with improved recall after Adjusting class proba.
